# Asteroid Game
## Learning Goals
The learning goals of this project are as follows:

- To introduce you to multi-file Python projects
- To get you used to referencing documentation (something you'll do often in the real world)
- To show you a real-world use case for object-oriented programming
- To have a ton of fun building a rewarding project!

Using pygame and a virtual environment to develop the game.

- python 3.13
- pygame 2.6.1
- [uv](https://docs.astral.sh/uv/)


Run project:
```shell
uv run main.py
```
